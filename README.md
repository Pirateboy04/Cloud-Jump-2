Cloud Jump 2
================

Expanding on the original [Cloud Jump] [1].

Additional features so far:
------
* Sound Effects
* Play in Portrait or Landscape Mode
* Player's High Score is saved
	* Player's name is also saved, and only needs to be entered once.
